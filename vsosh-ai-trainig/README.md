# ML Mini Projects

В этом репозитории собраны небольшие проекты по анализу данных и машинному обучению.  
Часть проектов была выполнена в рамках подготовки к **ВсОШ по искусственному интеллекту**.

## Проекты

### `vsosh-ai-training/netflix_eda.ipynb`
EDA и визуализация данных по каталогу Netflix: структура данных, пропуски, типы контента и основные наблюдения по датасету.

### `vsosh-ai-training/heart_disease_classification.ipynb`
Задача классификации на датасете Heart Disease: первичный анализ данных, подготовка признаков, обучение и сравнение моделей.

### `vsosh-ai-training/diamond_price_regression.ipynb`
Задача регрессии на датасете Diamonds: анализ данных, подготовка признаков и сравнение моделей для предсказания цены.

### `eda_students_performance.ipynb`
Разведочный анализ датасета Students Performance: базовые статистики, группировки и сравнение результатов между категориями.

### `eda_visualization_students.ipynb`
Визуализация данных Students Performance: распределения, зависимости между признаками и сравнение групп с помощью графиков.

### `user_behavior_analysis.ipynb`
Анализ пользовательского поведения на Stepik: feature engineering, построение baseline-модели и оценка качества классификации.

## Используемые инструменты

`Python`, `pandas`, `numpy`, `matplotlib`, `seaborn`, `scikit-learn`
