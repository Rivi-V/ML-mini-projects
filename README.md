# ML Mini Projects

Небольшие учебные проекты по анализу данных и машинному обучению.

## Содержание
- `vsosh-ai-training/` — проекты, выполненные в рамках подготовки к ВсОШ по ИИ
- `titanic_classification.ipynb` — классификация на датасете Titanic
- `Hybrid_Movie_Recommender.ipynb` — гибридная рекомендательная система для фильмов: popularity baseline, collaborative filtering, hybrid-подход и сравнение моделей по top-K метрикам
